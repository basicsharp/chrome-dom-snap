💡 Testing Tips:
  • Counter Test: Increment counter, take snapshot, change counter, restore → counter should return to snapshot value
  • Timer Test: Start timer, take snapshot, let it run, restore → timer should continue from snapshot time
  • Form Test: Fill out form, take snapshot, change values, restore → form should return to snapshot state
  • Scroll Test: Scroll to middle, take snapshot, scroll elsewhere, restore → should return to middle position
  • Focus Test: Focus input and type, take snapshot, click elsewhere, restore → input should regain focus
